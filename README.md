
# Installatation
* 1 - create a virtual environment and activate
*  - pip install virtualenv
*  - virtualenv envname
*  - envname\scripts\activate
* 2 - cd into project "cd freelancer"
* 3 - pip install -r requirements.txt
* 4 - python manage.py runserver
